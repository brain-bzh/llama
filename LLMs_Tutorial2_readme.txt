Tutorial 2
Preliminary steps to have it working:
    * Install the transformers library (pip install transformers)
    * Install the accelerate library (pip install accelerate)
    * Request Llama 2 download token from https://ai.meta.com/resources/models-and-libraries/llama-downloads/
    * Download the Llama 2 code from https://github.com/facebookresearch/llama and install it (pip install -e .)
    * Convert the Llama 2 model to a HuggingFace model (use the convert_llama_to_huggingface.py script from https://github.com/huggingface/transformers/blob/main/src/transformers/models/llama/convert_llama_weights_to_hf.py)
